# Service-Radar-Django-
Türkiye'deki niş ve zor bulunan hizmetler (örneğin, endüstriyel tamir, özel restorasyon, işlemci pin tamiri) ile bu hizmetleri sunan uzman firmaları güvenli, API odaklı bir platformda buluşturmak. Proje, satış işlemleri yerine güvenilir yönlendirme, entegrasyon ve komisyon takibi üzerine kurulacaktır.
